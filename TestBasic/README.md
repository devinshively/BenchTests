#webframeworks
=============

Basic example to get a feel of different backend web frameworks

##API:

Create/Update:  POST    /documents

List all:       GET     /documents

Delete:         DELETE  /documents/:id

Get:            GET     /documents/:id


##DB:

Postgresql

{id, title, text}
